# Components Directory

This directory contains reusable React components organized by feature and function.

## Structure

- `ui/` - shadcn/ui components and base UI primitives
- `layout/` - Layout components (Header, Sidebar, Footer, etc.)

## Usage

Import components using the `@/components` alias:

```tsx
import { Button } from '@/components/ui/button'
import { Header } from '@/components/layout/Header'
```
